Orness

Given a weighting vector, the orness gives us an idea of how similar an aggregation function is to the maximum function.

The ‘orness’ is generally used to characterise any averaging function (not just the OWA) and is named after the logical term “or”. For most averaging functions the ‘orness’ is difficult to calculate, but for the OWA the formula is simple:

∑i=1nwii−1n−1{"version":"1.1","math":"\\ \sum\limits_{i=1}^n w_i \frac{i-1}{n-1} \\"}

We can also interpret the orness as an indication of how much the function favours higher inputs.

  • If orness(w)=1{"version":"1.1","math":"\(\mathrm{orness}(\mathbf w) =1\)"}, then the OWA is the maximum with w=(0,0,…,0,1){"version":"1.1","math":"\(\mathbf w = ( 0,0,\ldots,0,1 )\)"}
  • If orness(w)=0{"version":"1.1","math":"\(\mathrm{orness}(\mathbf w) =0\)"},  then we obtain the minimum function. The orness of the arithmetic mean is 0.5{"version":"1.1","math":"\(0.5\)"}, however other weighting vectors like w=(0,0.5,0.5,0){"version":"1.1","math":"\(\mathbf w = (0, 0.5, 0.5, 0)\)"} or w=(0.1,0.2,0.4,0.2,0.1){"version":"1.1","math":"\(\mathbf w = (0.1, 0.2, 0.4, 0.2, 0.1)\)"} also have an orness of 0.5{"version":"1.1","math":"\(0.5\)"}. The orness is a concept that applies to all averaging functions (the geometric mean when n=2{"version":"1.1","math":"\(n=2\) "} is 1/3{"version":"1.1","math":"\(1/3\)"} while the harmonic mean (when n=2{"version":"1.1","math":"\(n=2\) "} has an orness of approximately 0.2274{"version":"1.1","math":"\(0.2274\)"}).

Example: Calculating the orness

Calculate the orness for an OWA with the weighting vector w=(0.1,0.5,0.4){"version":"1.1","math":"\(\mathbf w = ( 0.1, 0.5, 0.4 )\)"}.

Since n=3{"version":"1.1","math":"\(n = 3\)"}, the multipliers for the weights will be

(1-1)/(3-1)=0, (2-1)/(3-1)=1/2, (3-1)/(3-1)=1.

Therefore

orness((0.1,0.5,0.4))=0.1(0)+0.5(12)+0.4(1)=0.25+0.4=0.65.{"version":"1.1","math":"\mathrm{orness}(( 0.1, 0.5, 0.4 )) = 0.1(0) + 0.5(\frac{1}{2}) +0.4(1) =0.25 + 0.4 = 0.65."}

Since this is greater than 0.5{"version":"1.1","math":"\(0.5\)"}, we would say that this particular OWA tends more toward higher inputs.

Activity

A basic implementation of the orness function in R is given below. Implement it and play around with various weights with arrays of size 3 and size 4. e.g. Starting with c(0.2, 0.4, 0.2), by how much does the orness change if I move 0.1 to either side e.g. c(0.2, 0.3, 0.3)?

orness.OWA = function(w) {
n = length(w)
sum(w*(1:n-1)/(n-1))
}
