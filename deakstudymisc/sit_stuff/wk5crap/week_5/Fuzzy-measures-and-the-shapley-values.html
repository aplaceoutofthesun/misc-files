Fuzzy Measures and the Shapley Values

Fuzzy Measures

A fuzzy measure allocates a weight to a set of inputs with the following properties:

  • the full set has a value of one,

  • the empty set ∅{"version":"1.1","math":"\(\emptyset\)"} has a value of zero, and
  • if we add an element to the set, we can’t decrease the measure (i.e. a kind of monotonocity in terms of adding elements to the set).
The number of sets depends on the number of variables: if we have n{"version":"1.1","math":"\(n\)"} variables, then we have 2n{"version":"1.1","math":"\(2^n\)"} sets.  In the following example we have 3 variables, and so we have 23=8{"version":"1.1","math":"\(2^3 = 8\)"} different sets of which we assign a value to. 

Example: Projected percentages

For 3 workers, the following values represent the percentage of a project they can achieve in a day in each combination.

v({1,2,3})=1{"version":"1.1","math":"\\ v(\{1,2,3\}) = 1 \\ \\"}

v({1,2})=0.6v({1,3})=0.8v({2,3})=0.7{"version":"1.1","math":"\\ v(\{1,2\}) = 0.6 \quad v(\{1,3\}) = 0.8 \quad v(\{2,3\}) = 0.7 \\"}

v({1})=0.3v({2})=0.3v({3})=0.7{"version":"1.1","math":"\\ v(\{1\}) = 0.3 \quad v(\{2\}) = 0.3 \quad v(\{3\}) = 0.7 \\"}

v(∅)=0{"version":"1.1","math":"\\ v(\emptyset) = 0 \\"}

For example, if all three workers work together, they can complete 100% of a project per day. If worker 1 and worker 2 work together without worker 3, they can complete 60% of a project per day. If worker 3 works alone, he can complete 70% of a project per day.

Notice that a weight is associated with every possible combination of inputs, rather than just a single input. It might be the case that two inputs together are worth more or less than their individual components.

Shapley Values

In order to make the fuzzy measure easier to interpret (especially for a large set), we can calculate the Shapley values to measure the average importance of each variable.

The Shapley value is a vector (ϕ1,ϕ2,…,ϕn){"version":"1.1","math":"\((\phi_1, \phi_2, \ldots, \phi_n)\)"} and is interpreted the same as w{"version":"1.1","math":"\(\mathbf w\)"} is interpreted for the WAM. To that end, the Shapley indices ϕi{"version":"1.1","math":"\(\phi_i\)"} have values that are greater than or equal to zero, and they all add up to 1.

The Shapley values can be calculated directly from a fuzzy measure v{"version":"1.1","math":"\(v\)"}. We leave the equation here for reference but won’t expect you to do any direct calculation. We will however generate Shapley values using an R script, and learn how to interpret them.

ϕ(i)=∑S⊆{1:n}∖{i}(n−|S|−1)!|S|!n!(v(S∪{i})−v(S)){"version":"1.1","math":"\\ \phi(i) = \sum_{S\subseteq \{1:n\} \backslash \{i\}} \frac{(n-|S|-1)!|S|!}{n!} (v(S \cup \{i\})-v(S)) \\"}
