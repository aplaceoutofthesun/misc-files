MAIA ANGELOVA TURKEDJIEVA: This assignment will assess your knowledge and skills to analyse data using aggregation functions in R. We provide for you the data file and the R tool. You will randomly select a specified number of the data following the R command given in the specification. Thus, you will all be working with different datasets. Your first task is to inspect the data and plot them using scatterplots and histograms. Use these plots to make conclusions about the data distributions of different variables. Once this step has been completed, you will select a specified number of variables that you will have the freedom to choose. Apply appropriate transformations to prepare data for analysis with the aggregation functions.

Use the R package that you will be provided with. And carefully read the descriptions in it. By now, you will know all commands. And all you will need to do is follow the instructions in the package. It will be important to use the appropriate aggregation functions to fit the data and learn the unknown parameters. Select the best-performing model for your prediction. With this selected model, use the learned parameters for four selected variables to predict the variable of interest. Once you have completed these functions in R, wrote a short report to summarise your findings and assess your predictions. Put all your results, plots, and tables in a single PDF file. Upload this file onto the Assessment folder in CloudDeakin.

Add to this folder an R file with your code and a text file with the transformed data. For this assignment, you need to upload three files only. Do not compress your files. Just upload what is requested in the format requested.
