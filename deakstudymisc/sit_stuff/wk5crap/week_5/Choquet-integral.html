Choquet Integral

The (discrete) Choquet integral is an averaging aggregation function that is calculated based on the values of the fuzzy measure.

Cv(x)=∑i=1nx[i](v({[i],[i+1],…,[n]})−v({[i+1],[i+2],…,[n]})){"version":"1.1","math":"\\ C_v(\mathbf x) = \sum\limits_{i=1}^n x_{[i]} (v(\{[i],[i+1],\ldots,[n]\}) - v(\{[i+1],[i+2],\ldots,[n]\})) \\"}

Calculating a Choquet integral

The notation [i]{"version":"1.1","math":"\([i]\)"}  means we arrange the inputs in lowest to highest. The formula seems complicated, so we’ll demonstrate a calculation:

Example 1: All in day’s work

Determine the Choquet integral for the input x=(0.8,0.3,0.4){"version":"1.1","math":"\(\mathbf x = (0.8, 0.3, 0.4)\)"}  (which represents the proportion of a day’s work that each worker is willing to put in) and the fuzzy measure:

v({1,2,3})=1{"version":"1.1","math":"\\ v(\{1,2,3\}) = 1 \\"}

v({1,2})=v({1,3})=0.6,v({2,3})=0.9,{"version":"1.1","math":"\\ v(\{1,2\}) = v(\{1,3\}) = 0.6, v(\{2,3\}) = 0.9, \\"}

v({1})=0.4,v({2})=0.1,v({3})=0.1{"version":"1.1","math":"\\ v(\{1\}) = 0.4, v(\{2\}) = 0.1, v(\{3\}) = 0.1 \\"}

  1.  Arrange the inputs in order (keep track of which position it belonged to), which gives us
x↗=(0.3,0.4,0.8){"version":"1.1","math":"\\ \mathbf{x}_\nearrow = (0.3, 0.4, 0.8) \\"}
  1.  Determine the weights to apply to each input.
  • For the first entry in the sorted array, x[1]=x2=0.3{"version":"1.1","math":"\(x_{[1]} = x_2 = 0.3\)"}, since 0.3 is the 2nd entry in the original array, the associated weight is v({1,2,3}){"version":"1.1","math":"\(v(\{1,2,3\})\)"}.
i.e. we start with the full set {1,2,3}{"version":"1.1","math":"\(\{1,2,3\}\)"} and subtract the worker associated with the lowest weight, which is worker 2.
  • To work out the weight allocated to x[2]=x3=0.4{"version":"1.1","math":"\(x_{[2]} = x_3 = 0.4\)"},  we start with v({1,3}){"version":"1.1","math":"\(v(\{1,3\})\)"} then remove the set that is leftover when 0.4 is removed. Since 0.4 is the 3rd entry in the original array, v({1,3})−v({1}){"version":"1.1","math":"\(v(\{1,3\}) - v(\{1\})\)"}.
  • The weight of the last one  x[3]=x1=0.8{"version":"1.1","math":"\(x_{[3]} = x_1 = 0.8\)"}.
The calculation of the Choquet integral now becomes:
0.3[v({1,2,3})−v({1,3})]+0.4[v({1,3})−v({1})]+0.8[v({1})]{"version":"1.1","math":"\\ 0.3\Big[v(\{1,2,3\}) - v(\{1,3\})\Big] + 0.4\Big[v(\{1,3\}) - v(\{1\})\Big] + 0.8\Big[v(\{1\})\Big] \\"}
=0.3(1−0.6)+0.4(0.6−0.4)+0.8(0.4)=0.3(0.4)+0.4(0.2)+0.8(0.4)=0.52{"version":"1.1","math":"% <![CDATA[ \begin{align} & = 0.3(1-0.6) + 0.4(0.6-0.4) + 0.8(0.4) \\\\ & = 0.3(0.4) + 0.4(0.2) + 0.8(0.4) \\\\ &= 0.52 \end{align} %]>"}

Notice that in the last line, the numbers in brackets effectively make up a weighting vector (0.4, 0.2, 0.4) whose entries sum up to 1.

The result of the Choquet integral can be interpreted as the amount of the project that can be completed if x{"version":"1.1","math":"\(\mathbf x\)"}  represents the time commitment of each worker and all workers start at the same time (and can’t return once they leave).

However, the Choquet integral is not just about modelling worker output. It is a function that allocates importance to every subset of inputs and can model interaction and redundancy. In some cases arguments may not be worth very much by themselves but in conjunction with others can be very important.

Example 2: Scholarly scores

We want to evaluate students for a scholarship based on their scores in Mathematics, Physics and English. One thing that we want to account for is that students who are good at maths are often also good at physics, so in combining their scores, we don’t necessarily want to `double count’ this ability. On the other hand, we want the importance of maths and physics to be higher than English. An example of a fuzzy measure that suits this preference can be:

v(maths,physics)=0.5,v(physics,english)=v(maths,english)=0.9{"version":"1.1","math":"\\ v(\mathrm{maths,physics}) = 0.5, v(\mathrm{physics,english}) = v(\mathrm{maths,english}) =0.9 \\"}

v(maths)=v(physics)=0.45,v(english)=0.3{"version":"1.1","math":"\\ v(\mathrm{maths}) = v(\mathrm{physics}) = 0.45, v(\mathrm{english}) = 0.3 \\"}

Three students’ scores are given as follows:

Student Maths Physics English
Sachim 18 16 10
Courtney 10 12 18
Wasim 14 15 15

In calculating the Choquet integral for all three students:

  • For Sachim, we have 10(1−0.5)+16(0.5−0.45)+18(0.45)=13.9{"version":"1.1","math":"\(10(1-0.5) + 16(0.5-0.45) + 18(0.45) = 13.9\)"}.
  • For Courtney, we have 10(1−0.9)+12(0.9−0.3)+18(0.3)=13.6{"version":"1.1","math":"\(10(1-0.9) + 12(0.9-0.3) + 18(0.3) = 13.6\)"}.
  • For Wasim, we have 14(1−0.9)+15(0.9−0.3)+15(0.3)=14.9{"version":"1.1","math":"\(14(1-0.9) + 15(0.9-0.3) + 15(0.3) = 14.9\)"}.

So we were able to assign higher value to maths and physics, but still reward all-round ability.

Special Cases

The discrete Choquet integral includes both the WAM and OWA (and therefore all special cases of the OWA) as special cases. In this case, any two inputs have no interaction or are additive.

v(S)=∑i∈Sv({i}){"version":"1.1","math":"v(S) = \sum\limits_{i \in S} v(\{i\})"}

For example, the fuzzy measure

v({1,2,3})=1{"version":"1.1","math":"\\ v(\{1,2,3\}) = 1 \\"}

v({1,2})=0.8     v({1,3})=0.5     v({2,3})=0.7

v({1})=0.3v({2})=0.5v({3})=0.2{"version":"1.1","math":"\\ v(\{1\}) = 0.3 \quad v(\{2\}) = 0.5 \quad v(\{3\}) = 0.2 \\"}

v(∅)=0{"version":"1.1","math":"\\ v(\emptyset) = 0 \\"}

is equivalent to having a WAM with weights w=<0.3,0.5,0.2>{"version":"1.1","math":"\(% <![CDATA[ \mathbf{w}=<0.3,0.5,0.2> %]] >\)"}. Notice that v({1,2,3})=v({1})+v({2})+v({3}){"version":"1.1","math":"\(v(\{1,2,3\}) = v(\{1\}) + v(\{2\}) + v(\{3\})\)"} and that all interactions are additive, such as v({1,2}) =v({1}) +v({2}) and v({1,2,3}) =v({1,2}) +v({3}) .

  • OWA if v(S)=v(T){"version":"1.1","math":"\(v(S) = v(T)\)"} whenever |S|=|T|{"version":"1.1","math":"\(\vert S \vert = \vert T \vert\)"}.  i.e. if each subset has the same number of elements, their weights have the same measure.
