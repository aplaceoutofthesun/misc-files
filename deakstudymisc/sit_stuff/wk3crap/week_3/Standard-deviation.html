Standard deviation

When plotting data, some datasets are more clumped up together whilst others appear much more spread out.

The standard deviation is a single number that allows us to discern how spread out our data is. The variance and standard deviation are technically not aggregation functions as they do not satisfy the boundary properties. However they do share the same feature in that they take in a set of data and use a single value to express a feature of that data.

Standard deviation in practice

The higher the standard deviation, the higher the spread. Consider the two following histograms:

 Normal Distribution with μ=0{"version":"1.1","math":"\(\mu = 0\)"} and σ=3{"version":"1.1","math":"\(\sigma = 3\)"}

We can see the first histogram represents data that are more tightly clumped up towards the middle, whereas the second histogram is a bit more spread out. The first histogram therefore has a smaller standard deviation than the second histogram.

Mathematically, given a set of values x1,x2,…,xN{"version":"1.1","math":"\(x_1, x_2, \dots, x_N\)"} the (population) standard deviation (σ{"version":"1.1","math":"\(\sigma\)"}) is

σ=1N∑i=1N(xi−μ)2{"version":"1.1","math":"\\ \sigma = \sqrt{ \frac{1}{N} \sum_{i=1}^N(x_i - \mu)^2 } \\"}

where 

  • μ{"version":"1.1","math":"\(\mu\)"} is the (arithmetic) mean of x1,x2,…,xN{"version":"1.1","math":"\(x_1, x_2, \dots, x_N\)"}.
  • N{"version":"1.1","math":"\(N\)"} is the number of values.

    If we are to manually calculate the standard deviation of a set of numbers 1,4,7,12{"version":"1.1","math":"\(1,4,7,12\)"}.

  1.  We’ll want to calculate the mean first:
    μ=1+4+7+124=6{"version":"1.1","math":"\\ \mu = \frac{1+4+7+12}{4} = 6 \\"}
  2.  We then calculate the difference between each of our observed values and the mean, then square those differences so that all our values are positive.
    Note now that all these values here can be considered an ‘objective’ measure of how far each data point is from the mean, which represents the spread of data.
    xi{"version":"1.1","math":"x_i"} 1{"version":"1.1","math":"1"} 4{"version":"1.1","math":"4"} 7{"version":"1.1","math":"7"} 12{"version":"1.1","math":"12"}
    μ−xi{"version":"1.1","math":"\mu-x_i"} 5{"version":"1.1","math":"5"} 2{"version":"1.1","math":"2"} −1{"version":"1.1","math":"-1"} −6{"version":"1.1","math":"-6"}
    (μ−x1)2{"version":"1.1","math":"(\mu-x_1)^2"} 25{"version":"1.1","math":"25"} 4{"version":"1.1","math":"4"} 1{"version":"1.1","math":"1"} 36{"version":"1.1","math":"36"}
  3. We now want the average of these measures: we add up all the prior values and divide by the number of values. What we have now is the variance σ2{"version":"1.1","math":"\(\sigma^2\)"}.
    σ2=14(25+4+1+36)=16.5{"version":"1.1","math":"\\ \sigma^2 = \frac{1}{4} (25+4+1+36) = 16.5 \\"}
  4.  Take the square root of our variance to obtain our standard deviation.
    σ=16.5≈4.062{"version":"1.1","math":"\\ \sigma = \sqrt{16.5} \approx 4.062 \\"}

Note: Variance is not explicitly addressed it does not have as much relevance to the content in these weeks. The scenarios in which one would address/use the variance is different to the scenarios where we’d prefer the standard deviation.

In addition to this, there is a distinction to be made between population standard deviation and sample standard deviation, where in most scenarios we’re taking a sample from a population. The formula is slightly different, where we divide by N−1{"version":"1.1","math":"\(N-1\)"} instead of N{"version":"1.1","math":"\(N\)"} since it is mathematically proven to give a better estimate.

σ=1N−1∑i=1N(xi−μ)2{"version":"1.1","math":"\\ \sigma = \sqrt{ \frac{1}{N-1} \sum_{i=1}^N(x_i - \mu)^2 } \\"}

We won’t investigate the details, but point out that the built in standard deviation sd() function in R uses this formula. To test against our earlier example, sd(c(1,4,7,12)) should give approximately 4.69{"version":"1.1","math":"\(4.69\)"}.
