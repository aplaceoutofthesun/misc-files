Median and Mode

The median and the mode are less common measures of central tendency which can still provide quick sufficient insights as needed.

Median

The median of a set of data is the ‘middlest’ number after the data is ordered. For example if I have a set of values 5,9,3,1,4{"version":"1.1","math":"\(5,9,3,1,4\)"}, the first step would be to organise them in ascending order:

1,3,4,5,9{"version":"1.1","math":"1,3,4,5,9"}

and the median of these numbers would be 4{"version":"1.1","math":"\(4\)"} as it is the number in the middle.

In the case that we have an even set of numbers, we take the average of the middle two numbers. For example, given a set of ordered numbers

2,5,9,10,11,13{"version":"1.1","math":"2,5,9,10,11,13"}

there is no one number in the middle. As such, we take the average of the middle two numbers, giving us 9+102=9.5{"version":"1.1","math":"\(\frac{9+10}{2} = 9.5\)"}.

Mode

The mode of a set of values is the number which occurs the most often in a set of data. e.g. the mode of 1,1,2,2,3,3,3,4,5,6,6,7,8{"version":"1.1","math":"\(1, 1, 2, 2, 3, 3, 3, 4, 5, 6, 6, 7, 8\)"} is 3{"version":"1.1","math":"\(3\)"}. If we have multiple numbers with the same number of occurrences, we can equally consider them all modes of the given set of data. e.g. the mode of 11,11,13,14,15,15{"version":"1.1","math":"\(11, 11, 13, 14, 15, 15\)"} is both 11{"version":"1.1","math":"\(11\)"} and 15{"version":"1.1","math":"\(15\)"}.

If there are no repeated numbers in the given data set, then there is no mode.

Activity

Every time you encounter a new measure, method or model consider the following questions. This is a fundamental way of getting a deeper understanding of those measures/methods/models and the scenarios that they are best applied to.

The questions in this task are paraphrasings of more general questions:

  • In what scenario or data set would measure X{"version":"1.1","math":"\(X\)"} be better than the measure Y{"version":"1.1","math":"\(Y\)"}?
  • In what scenario or data set would measure Z{"version":"1.1","math":"\(Z\)"} be useful or useless?

Alternatively:

  • What data sets work with measure X{"version":"1.1","math":"\(X\)"}? Which data sets don't work as well and why?

Respond to one or more of the suggested questions (or make up your own) as a comment below.

  • What type of data would the median not be sufficient for measuring?
  • What kind of data would we be looking at such that the mode would not be a sufficient measure of central tendency?
  • What kind of data distribution would we have in which the mean, median and mode would all give the same value?
  • What would a data distribution look like that would favour using the median over the mode, or the mode over the median?
  • In what cases would the median and/or mode be a better measure of central tendency than the arithmetic mean?

Extension: Write your own functions in R to calculate the median and mode of an array of numbers. The official functions in R are median and modeest to compare your results with. Note that R does not have a standard in-built function to calculate the mode. There are many functions in the package of "modeest" for mode estimation under different data distribution, please see the documentation for detail. You can use the function "mfv(x)" to get the mode in a numerical vector x.  An example is:

install.packages("modeest")  # delete this line if you have already installed it
library(modeest)
mfv(c(1,1,1,2,1,3))
