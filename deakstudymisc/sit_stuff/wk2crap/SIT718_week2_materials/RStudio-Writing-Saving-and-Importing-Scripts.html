RStudio: Writing, Saving and Importing Scripts

Script files are an essential component of your R programming arsenal. They are useful for storing commands and functions that you’d like to use in future.

Watch Matt Mack in this short tutorial that shows how to store functions in scripts and load them when you need them again. This function was covered briefly in week 1, so this reintroduction has been provided as a refresher. Once you are confident with the navigation apply what you have learned by completing the following programming tasks.

Create a new script (Ctrl+Shift+N) to save our functions in. We can also have multiple functions in a single script file as it will allow us to load all functions at once in future. Ideally by the end of the week we’ll have one or more script files that contain the functions we’ve programmed and implemented.

The general structure of a function is as such:

name.of.function = function(input1, input2) {
  # Comment describing what the function does, or explaining a step in calculations.
  # Sequence of calculations
}
  • name.of.function - the name of a function can be any variable name and should be descriptive.
  • input1, input2 - a function can have one or many inputs and they can be of any data type. When writing the function it should be clear as to what form you intend the inputs to be: a number, array or string.
  • # Comments - comments can be written in your code as long as they start with a hash sign #. It is recommended to insert comments in your code that explain any aspect of the code that is not immediately familiar to you or anyone reading it. You will notice that RStudio will colour comments differently.
  • # Sequence of calculations - the calculations happening inside of a function that depend on the inputs given.

A simple implementation of the sum function which adds up all the numbers in an array can be written as follows.

sumArray = function(arrayOfNumbers) {
    result = 0
    for (i in range(arrayOfNumbers)) {
        result = result + arrayOfNumbers[i]
    }
    result
}
