Log & Polynomial Transformations

Some scatterplots will show data appearing on a curve, or data clumping up or skewing towards one end instead of being spread out normally.

A log or polynomial transformation can transform the data into something more linear and easier to look at.

Log Transformations

Another common technique in statistical regression is to transform the data using functions like:

xnew=ln⁡(xold){"version":"1.1","math":"\(\\ x_{\mathrm{new}} = \ln (x_{\mathrm{old}})\)"} and xnew=(xold)2{"version":"1.1","math":"\(x_{\mathrm{new}} = (x_{\mathrm{old}})^2\)"}

  • Such functions can help if data are exponentially distributed or skewed.
  • The log function in particular is useful for data that can have a few very large inputs .
  • Incomes, populations in ecology and academic journal citations are all examples of data that could exhibit such properties.

Polynomial Transformations

  • Polynomial functions t2{"version":"1.1","math":"\(t^2\)"} or t12{"version":"1.1","math":"\(t^{\frac{1}{2}}\)"}) can have a similar (but less dramatic) affect, allowing skewed distributions to look more like normal distributions.
  • For tp{"version":"1.1","math":"\(t^p\)"}, 0<p<1{"version":"1.1","math":"\(0 < p < 1 %\)"},  can be used when there are fewer very high values (positive skew), while  p>1{"version":"1.1","math":"\(p > 1\)"} when there are fewer very low values (negative skew). This transforms work well when the data is already on the unit interval.
