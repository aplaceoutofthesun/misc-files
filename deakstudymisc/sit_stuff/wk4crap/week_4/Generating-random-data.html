Generating random data

There are times where we want to experiment or test certain methods and analyses.

One way to do this, is to test them on a dataset that is generated according to a known distribution. This allows us to confirm any hypotheses against a dataset whose behaviour we are familiar with.

The aim of this article is to show you how to generate random data according to specific distributions, and plot their associated histograms. You can then use this knowledge to practice applying transformations to this generated data and plotting the histogram (and/or scatterplot) after each transformation to see how your data changes.

For a full list of distributions supported in R’s base packages, see the topic, “Distributions in the stats package”, in the R Documentation .

Uniform Distribution in R

Looking at the documentation, the function for generating random numbers in a uniform distribution is runif(n, min = 0, max = 1) where n is the number of values that we want to generate, min is the lowest number and max is the highest number.

For example, if we want to randomly generate 1000{"version":"1.1","math":"\(1000\)"} numbers in a uniform distribution between −2{"version":"1.1","math":"\(-2\)"} and 4{"version":"1.1","math":"\(4\)"} and assign this to a variable called uniformdata, we would type:

uniformdata = runif(1000,-2, 4)
  • Type uniformdata or head(uniformdata) to view the data itself.
    • What do you notice?
  • Type hist(uniformdata) and take a look at the histogram.
    • Does it look like a uniform distribution?
    • If it doesn’t, why?

Normal Distribution in R

Similarly for the normal distribution, the documentation states that the function for generating numbers in a normal distribution is done by the function

rnorm(n, mean = 0, sd = 1)

For example, if we want to randomly generate 9999 numbers in a normal distribution with mean5{"version":"1.1","math":"\(5\)"} and standard deviation2{"version":"1.1","math":"\(2\)"} and assign it to a variable normaldata, then we would type:

normaldata = rnorm(9999, 5, 2)
  • Like before, type hist(normaldata) to view the histogram.

Activity

What happens if we only enter in one argument for each of runif and rnorm e.g. runif(100)?

You can manipulate how many columns your histogram shows by specifying how many columns/bins you want to show (e.g. hist(uniformdata, 100)) or by specifying a vector of intervals (e.g. hist(uniformdata, seq(-2, 4, 0.1))).
